# 🔗 URL Shortener

A simple and efficient URL Shortener built with Node.js, Express, and MongoDB.

Convert long, messy links into short, shareable URLs in seconds.

🌍 **Live Demo:**  
https://url-shortener-1-qksx.onrender.com/

---

## ✨ Features

- 🔐 Generates unique short URLs
- ⚡ Instant redirection to original links
- 🗄️ MongoDB database storage
- 🎨 Simple and clean UI
- 🚀 Deployed on Render

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Frontend:** EJS + Bootstrap  
- **Deployment:** Render
